adjust_grid
===========
Choose whether or not you would like to adjust the scale length
for the logarithmic grid. Advanced command. 

**Type:** Boolean (1/0)

**Parent(s):**
  parameter_: Condition e.g. greater than 0 or list e.g. [1, 2, 5]


**File:** setup_domains.c
