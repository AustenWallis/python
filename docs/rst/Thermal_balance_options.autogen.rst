Thermal_balance_options
=======================
This is a paramerter designed to turn off some heating/cooling 
mechanisms. At the moment, all it does is turn off adiabatic 
effects - typically used in hydrodynamic simulations.

**Type:** Enum (Int)

**Values:**

0. everything.on

1. no.adiabatic

  parameter_: None
